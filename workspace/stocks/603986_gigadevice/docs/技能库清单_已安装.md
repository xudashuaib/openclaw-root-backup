## 使用建议

### 预测任务触发顺序
1. **数据获取**: `allstock-data` → 获取实时行情 + K线
2. **技术分析**: `chartclass` → 识别K线形态
3. **技术指标**: `china-stock-analysis` → 均线/MACD/RSI
4. **宏观分析**: `yumstock` → 全球市场情绪
5. **深度研究**: `deep-research-pro` → 搜索最新消息/公告
6. **数据分析**: `Data Analysis` → 财务数据可视化
7. **综合研判**: `mental-models` + `trading-knowledge` → 整合判断

### 自学习机制
- 每次复盘后调用 `self-improvement` 记录经验
- 预测偏差大时调用 `Self-Improving + Proactive Agent` 触发模型优化
